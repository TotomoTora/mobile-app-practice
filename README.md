# Мобильное приложение для практики

## Описание
Разработка мобильного приложения для Android с использованием Supabase в качестве бэкенда.

## Требования
- Android 13.0 (API 33) и выше
- Supabase в качестве бэкенда

## Архитектура
Проект использует чисту� архитектуру с разделением на слои:
- Data Layer (работа с данными)
- Domain Layer (бизнес-логика) 
- Presentation Layer (UI)
